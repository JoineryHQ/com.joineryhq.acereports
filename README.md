# com.joineryhq.acereports

Custom reports for ACE.

This is an [extension for CiviCRM](https://docs.civicrm.org/sysadmin/en/latest/customize/extensions/), licensed under [GPL-3.0](LICENSE.txt).

## Getting Started

Install, and check New Report for report templates marked as provided by this extension.
